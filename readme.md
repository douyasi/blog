## Typecho博文导出脚本使用说明

本导出脚本只支持使用 `markdown` 编辑的文章。

请自行修改 `Typecho` 数据库与站点配置等参数，然后在命令行执行 `php export.php` ，导出代码见 `typecho.php` ，源码依赖于 `medoo` 数据库框架。
